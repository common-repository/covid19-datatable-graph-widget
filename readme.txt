Coronavirus disease (COVID-19) is an infectious disease caused by a new virus. The plugin allows adding statistics table/widgets via shortcode to inform your site visitors about changes in the situation about Coronavirus pandemic.

== Description ==

This plugin displays the Coronavirus ( Covid19 ) case data of the whole world and country you care through a shortcode in your WordPress post or page. You can use this shortcode with some attributes also. You can also able to show the data as graph. This plugin also allows you to add Data Ticker to widgets via shortcode. Also inline text data is available.

= The following shortcode options are available =
-> Widget Shortcode
-> List of Countries
-> Graph
-> Table of Countries,
-> Data Ticker
-> Inline Text data
-> Map of Countries (Pro Feature)
-> Map of USA (Pro Feature)

= Requirements =
Website with https (SSL certificate)
WordPress 4.6.x or above
Available WP-cron (enabled by default)

= What are the sources of data informing the dashboard? =

This plugin is relying on a 3rd party as a service for Realtime Covid19 Data.
The link of the third party serviec is: https://corona.lmao.ninja/
Privacy Policy link: https://github.com/NovelCOVID/API/blob/master/privacy.md

The data sources include the World Health Organization, the U.S. Centers for Disease Control and Prevention, the European Center for Disease Prevention and Control, the National Health Commission of the People’s Republic of China, and the DXY, one of the world’s largest online communities for physicians, health care professionals, pharmacies and facilities.


= How often is the data updated? =
Data is updated every 10 minutes.


== Installation ==
1. Download the plugin zip file. Extract and upload in your wp-content/plugins folder.
2. From the wp-admin panel go to plugins and activate "Covid19 DataTable Graph & Widget"
4. You are done.


== Use ==
1. After activating the plugin you will see a menu called Covid19 DataTable.
2. Click on the menu and to get all the available shortcodes.
3. Then add the shortcodes to page or post to display.
